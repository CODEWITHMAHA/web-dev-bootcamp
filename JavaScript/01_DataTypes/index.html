Data Types

Primitive Data Types

  • String: A sequence of characters, e.g., "Hello, World!"
  • Number: Numeric values, e.g., 42, 3.14
  • Boolean: Represents true or false values.
  • Null: Represents the intentional absence of any object value.
  • Undefined: A variable that has not been assigned a value.
  • Symbol: A unique and immutable primitive value used as the key of an object property.
  • BigInt: Represents integers with arbitrary precision.

Non-Primitive Data Types

  • Object: A collection of properties, where each property is a key-value pair.
  • Array: An ordered list of values, which can be of mixed data types.
  • Function: A block of code designed to perform a particular task.

Question 1: What are the different data types in JavaScript?

‚úÖ Answer: JavaScript has 8 data types ‚Äî 7 primitive types and 1 non-primitive (object) type.

Primitive types:
  • String: represents textual data ("Hello")
  • Number: represents numeric values (42, 3.14)

  • Null: represents intentional absence of value

Non-primitive type:

    Question 3: What is the difference between primitive and non-primitive data types?

    Answer: Primitive types are immutable and stored by value.Each variable holds its own copy.

            let a = 10;
        let b = a;
        b = 20;
        console.log(a); // 10

    Non-primitive types (objects) are mutable and stored by reference.

            let obj1 = { name: "Maha" };
        let obj2 = obj1;
        obj2.name = "Khan";
        console.log(obj1.name); // Khan
