String Methods And Properties

Strings in JavaScript come with a variety of built-in methods and properties that allow you to manipulate and interact with text data easily. Here are some commonly used string methods and properties:

Common String Methods

  • charAt(index): Returns the character at the specified index.
  • concat(string1, string2, ...): Joins two or more strings and returns a new string.
  • includes(substring): Checks if the string contains the specified substring and returns a boolean.
  • indexOf(substring): Returns the index of the first occurrence of the specified substring, or -1 if not found.
  • slice(startIndex, endIndex): Extracts a section of the string and returns it as a new string.
  • toLowerCase(): Converts the entire string to lowercase letters.
  • toUpperCase(): Converts the entire string to uppercase letters.
  • trim(): Removes whitespace from both ends of the string.

Common String Properties

  • length: Returns the length of the string (number of characters).

Example Usage

    
      let str = "  Hello, World!  ";
      console.log(str.length); // 17
      console.log(str.trim()); // "Hello, World!"
      console.log(str.toUpperCase()); // "  HELLO, WORLD!  "
      console.log(str.indexOf("World")); // 9
      console.log(str.slice(2, 7)); // "Hello"

Methods without Arguments

Some string methods do not require any arguments. Here are a few examples:

  • toLowerCase(): Converts the entire string to lowercase letters.
  • toUpperCase(): Converts the entire string to uppercase letters.
  • trim(): Removes whitespace from both ends of the string.

Methods with Arguments

Many string methods require one or more arguments to function properly. Here are some examples:

  • charAt(index): Requires an index argument to specify which character to return.
  • charCodeAt: charCodeAt(index): Requires an index argument to return the Unicode of the character at that position.
  • replace: replace(oldSubstring, newSubstring): Requires two arguments: the substring to be replaced and the new substring.
  • replaceAll: replaceAll(oldSubstring, newSubstring): Requires two arguments: the substring to be replaced and the new substring.
  • padStart: padStart(targetLength, padString): Requires the target length and the string to pad with.
  • padEnd: padEnd(targetLength, padString): Requires the target length and the string to pad with.
  • concat(string1, string2, ...): Requires one or more string arguments to concatenate.
  • includes(substring): Requires a substring argument to check for its presence in the string.
  • indexOf(substring): Requires a substring argument to find its index in the string.
  • slice(startIndex, endIndex): Requires start and end index arguments to extract a portion of the string.
