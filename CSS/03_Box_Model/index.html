What is the CSS Box Model?

The CSS Box Model defines how every element on a webpage is structured and how its total size is calculated. Each element consists of four parts: content, padding, border, and margin.

The total element size = content + padding + border + margin.

By default, CSS uses the content-box model, but we can use box-sizing: border-box to include padding and border inside the defined width and height

Content

The content area is where text and images appear. It is the innermost part of the box model and its size can be defined using width and height properties in CSS.

Padding

Padding is the space between the content and the border. It creates space inside the element, pushing the content away from the edges.

Border

The border is the line that surrounds the padding (if any) and content. It can be styled with different widths, colors, and styles (solid, dashed, dotted, etc.).

Margin

Margin is the outermost layer that creates space between the element and other elements on the page. It is transparent and can be adjusted to control the distance between elements.
