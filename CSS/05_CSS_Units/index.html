CSS UNITS

CSS units are used to specify lengths, sizes, and other measurements in CSS. They can be broadly categorized into absolute and relative units.

Absolute Units

  • px (pixels): A pixel is a single dot on the screen. It is a fixed unit and does not change based on other factors.
  • cm (centimeters): A centimeter is a physical measurement. It is rarely used in web design due to varying screen sizes.
  • mm (millimeters): Similar to centimeters, millimeters are also physical measurements.
  • in (inches): An inch is another physical measurement, equivalent to 2.54 centimeters.
  • pt (points): A point is 1/72 of an inch, commonly used in print media.
  • pc (picas): A pica is equal to 12 points or 1/6 of an inch.

Relative Units

  • em: The em unit is relative to the font-size of the parent element. For example, 2em means twice the size of the current font.
  • rem (root em): The rem unit is relative to the font-size of the root element (usually the <html> element).
  • % (percentage): Percentage units are relative to the parent element's size. For example, a width of 50% means half the width of the parent element.
  • vw (viewport width): 1vw is equal to 1% of the width of the viewport (the browser window).
  • vh (viewport height): 1vh is equal to 1% of the height of the viewport.
  • vmin: vmin is equal to 1% of the smaller dimension (width or height) of the viewport.
  • vmax: vmax is equal to 1% of the larger dimension (width or height) of the viewport.
