CSS Selectors

Q: What are selectors?

A: Selectors are used to "find" (or select) the HTML elements you want to style.

selector { property: value; }

p { color: blue; }

This selector targets all

elements and makes their text blue.

Types of CSS Selectors

Universal Selector

The universal selector (*) selects all elements on the page.

Example: * { margin: 0; }

Element Selector

Selects elements by their tag name.

Example: p { color: red; }

Class Selector

Selects elements with a specific class.

Example: .btn { background: blue; }

ID Selector

Selects a single element with a specific ID.

Example: #header { font-size: 20px; }

Attribute Selector

Selects elements by attribute or value.

Example: input[type="text"] { color: green; }

Group Selector

Applies same style to multiple selectors.

Example: h1, h2, h3 { font-family: Arial; }

Descendant Selector

Selects elements nested inside others.

Example: div p { color: gray; }

Child Selector

Selects direct child elements.

Example: ul > li { color: orange; }

Pseudo-class Selector

Selects based on a specific state.

Example: a:hover { color: red; }

Pseudo-element Selector

Styles specific parts of elements.

Example: p::first-line { font-weight: bold; }
